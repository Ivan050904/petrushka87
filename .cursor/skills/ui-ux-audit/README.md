# UI/UX Audit

Scored UI audit (0–100) and silent prevention while writing UI code.

## Source

https://github.com/EnchStyle/ui-ux-audit-skill

## How to use in Cursor

- **Full audit:** `Run a full UI audit on frontend/src/features/articles/articles-panel.tsx`
- **Quick review:** `Review spacing on this component` / `Why does this look off?`
- **Prevention:** automatic when editing React/HTML/CSS — agent consults this skill before shipping UI

## Modes

1. Full audit — 15 categories, scored report, ship verdict
2. Quick review — targeted scan
3. Prevention — pre-flight while coding

See `SKILL.md` for severity rubric and report format.
